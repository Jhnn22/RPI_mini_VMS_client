# RPI_mini_VMS_client

- basic requirements
```bash
# GStreamer 기본 설치
sudo apt install libgstreamer1.0-dev gstreamer1.0-tools gstreamer1.0-rtsp gstreamer1.0-plugins-base gstreamer1.0-plugins-good gstreamer1.0-plugins-bad gstreamer1.0-plugins-ugly gstreamer1.0-libav libgstrtspserver-1.0-dev

# libcamera 관련 라이브러리 설치(아래의 코드에서 libcamerasrc를 사용할 때 필요하다)
sudo apt install libcamera-dev gstreamer1.0-libcamera

```
